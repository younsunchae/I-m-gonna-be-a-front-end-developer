1. list를 나열할 때
2. list의 인덱스를 지정해줄 때 ex)mise[0], mise[1]

## 📌 {} 📌
1. dictionary를 만들 때
```jsx
{`key` : `value`}
```
2. style tag를 적용해 줄 때 
 ```jsx 
.styleforMobile { ***적용할 것들*** }
 ```
3. 특정function 에 대해 정의해줄 때 
```jsx
.function openClose() { ***실행할 것들*** }
```
4. for문의 조건문 뒤에 실행할 것을 적어 줄 때 
```jsx
for (int i=0; i<100; i++)
{ ***실행할 것들*** }
```
5. if문 뒤에 실행 할 것들을 적어 줄 때 
```jsx
if(a>b)
{ ***실행할 것들*** }
```
