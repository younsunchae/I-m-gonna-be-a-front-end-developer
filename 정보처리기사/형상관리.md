# 소프트웨어 패키징의 형상관리

형상관리(SCM; Software Configuration Management)는 소프트웨어 개발 과정에서 소프트웨어의 변경 사항을 관리하기 위해 개발된 일련의 활동

## 형상 관리 기능

- 형상 식별, 버전 제어, 형상 통제 , 형상 감사 , 형상 기록

## 소프트웨어 버전 등록 과정

- 가져오기(import) -> 인출(check out)-> commit -> Update -> Diff

## 형상관리 도구

### 공유 폴더 방식

SCCS, RCS, PVCS, QVCS

### 서버/클라이언트 방식

SVN, CVS

### 분산저장소 방식

GIT, DCVS, BAZZAR
