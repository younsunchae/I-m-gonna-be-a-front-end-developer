### 모듈
모듈화를 통해 분리된 시스템의 각 기능

### 결합도
모듈 간의 상호 의존하는 정도

#### 내용 결합도
content coupling <br />
한 모듈이 다른 모듈의 내부 기능 및 그 내부 자료를 직접 참조하거나 수정할 때의 결합도

#### 공통 결합도
common coupling <br />
공유되는 공통 데이터 영역을 여러 모듈이 사용할 때의 결합도
#### 외부 결합도
External coupling <br />
어떤 모듈에서 선언한 데이터를 외부의 다른 모듈에서 참조할 때의 결합도
#### 제어 결합도
Control coupling<br />
어떤 모듈이 다른 모듈 내부의 논리적이 흐름을 제어하기 위해 제어 신호나 제어 요소를 전달하는 결합도

####  스태프 결합도
stamp coupling<br />
모듈 간의 인터페이스로 배열이나 레코드 등의 자료 구조가 전달될 때의 결합도

#### 자료 결합도
Data coupling <br />
모듈 간의 인터페이스가 자료 요소로만 구성될 때의 결합도

### 응집도
모듈의 내부 요소들이 서료 관련 되어 있는 정도

#### 기능적 응집도
functional cohesion <br/>
모듈 내부의 모든 기능 요소들이 단일 문제와 연관되어 수행될 경우의 응집도
#### 순차적 응집도
sequential cohesion <br />
모듈 내 하나의 활동으로부터 나온 출력 데이터를 그 다음 활동의 입력 데이터로 사용할 경우의 응집도
#### 교환적 응집도
communication cohesion <br />
동일한 입력과 출력을 사용하여 서로 다른 기능을 수행하는 구성 요소들이 모였을 경우의 응집도
#### 절차적 응집도
Procedural cohesion <br />
모듈이 다수의 관련 기능을 가질 때 모듈 안의 구성 요소들이 그 기능을 순차적으로 수행할 경우의 응집도
